YENİLİKLER
- El mikrofonu, kürsü mikrofonu, yaka mikrofonu ve sunum kumandası için adet seçimi
- Artı / eksi butonları
- Video, ses sistemi, PowerPoint ve PDF seçenekleri
- Etkinlik provası seçeneği
- Hazırlanan e-postada ekipman adetlerinin açık şekilde gösterilmesi

GITHUB'A YÜKLEME
1. Eski index.html, styles.css ve app.js dosyalarını silmenize gerek yoktur.
2. Bu ZIP içindeki aynı adlı dosyaları GitHub'a yükleyin.
3. GitHub, aynı isimli dosyaları güncellemek için onay ister.
4. Commit changes butonuna basın.
5. GitHub Pages birkaç dakika içinde güncellenir.

- Etkinlik öncesinde bildirilmeyen teknik ihtiyaçlar için destek sağlanamayabileceğine dair uyarı eklendi.
